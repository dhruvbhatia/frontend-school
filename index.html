Frontend Developer interviews are challenging.

Frontend School is an open-source repository of common frontend interview puzzles.

We have an assortment of JavaScript, CSS, and HTML puzzles to help you prepare for interviews. Every puzzle contains an annotated solution to ensure you learn the best-practice way of doing things.

We are open-source so feel free to contribute by adding new puzzles or improving existing ones!
